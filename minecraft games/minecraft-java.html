info about "minecraft: java edition"

Minecraft: Java Edition (formerly just Minecraft) is the original version of Minecraft, developed by Mojang Studios for Windows, macOS, and Linux.
Notch began development on May 10, 2009, publicly releasing Minecraft on May 17, 2009.
The full release of the game (version 1.0.0) was on November 18, 2011, at MINECON 2011.
